title: AhoyDTU
description: AhoyDTU Widget
---

Learn more about [AhoyDTU](https://github.com/lumapu/ahoy).

Allowed fields: `["network", "nrf", "mqtt", "reboot_reason", "daily_reboot", "uptime", "inverterW*"]`.

`"inverterW*"` allows inverterW0 .. inverterW11.

```yaml
widget:
  type: ahoydtu
  url: http://ahoydtu.host.or.ip
```
